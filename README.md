# 数据结构

        逻结构是面向问题的，而物理结构就是面向计算机的，基本目的就是将数据及其逻关系存储到计算机的内存中。

### 集合结构

        集合结构中的数据元素除了同属于一个集合外，它们互相间没有其它的关系

### 线性结构
  
       线性结构中的数据元素之间是一对一的关系

### 树型结构
 
       树型结构中的数据源素之间是一对多的层次关系
       
### 图型结构

       图型结构中的数源素之间是多对多的层次关系

## 物理结构

      是指数据的逻结构在计算机中的存储形式
      
### 顺序存储结构

      数据元素放在地址连续的存储单元里，其数据间的逻辑关系和物理关系是一致的

### 链式存储结构
   
      数据元素存放在任意的存储单元里，这组存储单元可以是连续的，也可以是不连续的
