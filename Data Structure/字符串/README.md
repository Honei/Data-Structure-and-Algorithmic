# 串存储结构的具体实现

    存储一个字符串，数据结构包含以下 3 种具体存储结构：
     1. 定长顺序存储：实际上就是用普通数组（又称静态数组）存储。例如 C 语言使用普通数据存储字符串的代码为 char a[20] = "data.biancheng.net"；
     2. 堆分配存储：用动态数组存储字符串；
     3. 块链存储：用链表存储字符串；
